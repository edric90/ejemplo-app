Actualizando Git
Al realizar cualquier cambio en codigo, se puede verificar con la siguiente linea el estado de GitHub

Para actualizar el codigo: git add -A
luego realizar commit: git commit -m "Segunda actualizacion"
para cargar los cambios actualizar: git push origin main

Para descargar proyectos: git clon "url de proyecto"
